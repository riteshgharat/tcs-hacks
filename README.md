# Run

**Backend** (http://localhost:8787)

```powershell
cd backend; bun run dev
```

**Web app** (http://localhost:5173)

```powershell
cd web-app; bun run dev
```

**Chrome extension** — build, then Chrome → Extensions → Load unpacked → `chrome-ext/dist`

```powershell
cd chrome-ext; bun run build
```
